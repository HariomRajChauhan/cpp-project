# Lipi

## Building the Project

### Prerequisites

- CMake 3.10 or higher
- A C++ compiler (e.g., g++, clang++)

### Steps

1. **Clone the Repository**:
    ```sh
    git clone https://github.com/yourusername/lipi.git
    cd lipi
    ```

2. **Create a Build Directory**:
    ```sh
    mkdir build
    cd build
    ```

3. **Run CMake**:
    ```sh
    cmake ..
    ```

4. **Build the Project**:
    ```sh
    make
    ```
